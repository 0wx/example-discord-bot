## Installation
Clone this repository
```
git clone https://github.com/0wx/example-discord-bot.git
```
```
cd example-discord-bot
```
Install all dependencies
```
yarn
```

## Setup `.env` file for development
Rename `.env.example` to `.env`
then fill the value

## Run locally
Build the project
```
yarn build -w
```

and run
```
yarn dev
```
